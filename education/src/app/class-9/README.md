# Class 9 Route

This folder owns the Class 9 landing page and its future Class 9 route-specific UI assets.

- `page.tsx` renders the Class 9 subject overview.
- `Class9.module.css` stores only Class 9 page styles.
- `[subject]/page.tsx` renders working subject detail routes through the shared learning-plan component.

Keep Class 9 content, styling, and future folder expansion separate from other academic classes.
