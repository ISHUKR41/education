# Class 10 Route

This folder owns the Class 10 landing page and its future Class 10 route-specific UI assets.

- `page.tsx` renders the Class 10 board-preparation overview.
- `Class10.module.css` stores only Class 10 page styles.
- `[subject]/page.tsx` renders subject detail routes for the Class 10 catalog.

Keep Class 10 board content, styling, and related learning flows isolated from other classes.
